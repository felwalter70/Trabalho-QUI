o jogo de cartas

Sobre o jogo:

Chemistry é um jogo de cartas em turnos sobre química, cujo o objetivo é fazer mais pontos a partir de moléculas, o intuito do jogo é estimular os jogadores a aprenderem química de uma maneira simples e divertida.


-Regras Do Jogo-

Chemistry é baseado em turnos, com X no total, o objetivo é juntar pontos a partir das cartas dos elementos quimicos:

Ao lado, vemos a carta do elemento Ferro, no jogo, cada carta possui pontos que são determinados pelos elétrons na camada de valência, no exemplo, o Ferro possui 2 elétrons na camada de valência, logo, essa carta vale dois pontos
